# PDS Diary - Complete Feature List

## What's New in Your Year-Long Diary

Your PDS Diary has been transformed into a comprehensive year-long journaling system with advanced features for tracking, reflecting, and achieving your goals throughout the year.

### 1. Year Overview Dashboard (`/year`)
**Access:** Click "Year Overview" button on the home page

**Features:**
- **Interactive Calendar View**
  - Navigate through any month/year
  - Visual indicators for days with entries (blue gradient)
  - Click any day to view/edit entry or create new one
  - "Today" button for quick navigation
  - Previous/Next month navigation

- **Statistics Cards**
  - Total entries for the selected year
  - Current streak (consecutive days of journaling)
  - Longest streak all-time
  - Completion rate (entries with all three PDS sections filled)

- **Monthly Distribution Chart**
  - Bar chart showing entries per month
  - Gradient visualization of activity
  - Identify your most active months

- **Top Tags**
  - See your most-used tags with counts
  - Visual tag cloud with frequency indicators
  - Up to 20 top tags displayed

- **PDS Completion Statistics**
  - Separate completion rates for Plan, Do, and See
  - "All Three" metric for fully completed entries
  - Color-coded progress indicators

- **Year Selector**
  - Easily switch between different years
  - Automatically detects year range from your entries

### 2. Goals & Milestones System (`/goals`)
**Access:** Click "Goals" button on the home page

**Features:**
- **Create Different Goal Types**
  - Yearly goals (big picture, annual aspirations)
  - Quarterly goals (3-month objectives)
  - Monthly goals (short-term targets)

- **Goal Management**
  - Title and detailed description
  - Target date (optional)
  - Progress slider (0-100%)
  - Visual progress bar with gradient
  - Status tracking (active/completed/archived)

- **Milestones System**
  - Break down goals into smaller milestones
  - Checkbox tracking for each milestone
  - See completion count at a glance
  - Automatic completion timestamps

- **Organized Views**
  - Goals grouped by type (yearly/quarterly/monthly)
  - Separate section for completed goals
  - Edit and delete functionality
  - Inline editing with live preview

### 3. Reflection Templates
**Access:** When creating/editing any entry

**Four Template Types:**

**Daily (Default)**
- Standard Plan/Do/See prompts
- 24-hour time block tracking
- Perfect for everyday journaling

**Weekly**
- Structured weekly review prompts
- Top 3 priorities for the week
- Habit tracking focus
- Pattern recognition prompts
- "What worked well?" reflections

**Monthly**
- Deep-dive monthly reflection
- Goal categories (professional, personal, health, relationships, learning)
- Comprehensive accomplishment tracking
- Time allocation analysis
- Forward-looking planning

**Yearly**
- Annual vision setting
- Multi-dimensional goal areas
- Year-in-review structure
- Top wins and lessons
- Growth reflection
- Gratitude and advice to future self

**How to Use:**
1. Create or edit an entry
2. Click on the desired template (Daily/Weekly/Monthly/Yearly)
3. Template automatically fills Plan/Do/See sections with structured prompts
4. Customize and fill in your reflections

### 4. Enhanced Export System

**Export JSON** (existing, enhanced)
- Complete data backup
- All entries with full metadata
- Import to sync across devices

**Year Report (NEW)**
- Click "Year Report" button on home page
- Generates comprehensive Markdown report
- Includes:
  - Overview statistics
  - Entry type breakdown
  - Completion rates
  - Time block statistics
  - ASCII-style monthly distribution chart
  - Top tags list
  - All weekly/monthly/yearly reflections
  - Full yearly reflection content
- Perfect for end-of-year review
- Save and share your year's journey

## ðŸ’¡ Usage Tips

### For Best Results

1. **Start with Daily Entries**
   - Build the habit with simple daily reflections
   - Use time blocks to track your day
   - Add tags to make entries searchable

2. **Weekly Reviews**
   - Every Sunday/Monday, create a weekly reflection
   - Review what worked and what didn't
   - Set top 3 priorities for the coming week

3. **Monthly Deep Dives**
   - First day of each month, do a monthly review
   - Look at patterns in your tags and entries
   - Adjust goals based on progress

4. **Yearly Planning**
   - Start of year: Create yearly vision entry
   - Set yearly goals in the Goals page
   - End of year: Export year report for review

5. **Goals System**
   - Link yearly goals to quarterly milestones
   - Break quarterly goals into monthly actions
   - Update progress weekly
   - Celebrate completed milestones!

### Workflow Suggestions

**Morning Routine:**
- Open entry for today
- Fill out "Plan" section
- Review time blocks, plan your day

**Evening Routine:**
- Fill out "Do" section (what actually happened)
- Complete time blocks
- Write "See" insights and learnings

**Weekly (Sunday):**
- Create weekly reflection entry
- Review the past week's entries
- Check Goals page, update progress
- Plan next week's priorities

**Monthly (1st of month):**
- Create monthly reflection entry
- Check Year Overview stats
- Update monthly goals
- Review tag patterns

**Yearly (Jan 1 / Dec 31):**
- Create yearly reflection entry
- Export year report
- Set new yearly goals
- Review completion of past goals
